# MangoFeed

MangoFeed is a fruit marketplace where users can:

- Buy fruits
- Order fruit juice
- Order fruit chaat
- Explore different mango varieties

## Features

- Authentication
- Product Management
- Categories
- Cart
- Orders
- Admin Dashboard
